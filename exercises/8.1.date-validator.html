Date Validator

		Description:
		Write a program to validate the given date (date-month-year) i.e., check whether the given date is valid one or not.
		
		Example 1:
		Output Screen:
		Enter date: 4-7-2015
		It is a valid date.

		Example 2:
		Output Screen:
		Enter date: 31-4-2015
		It is not a valid date.

		Example 3:
		Output Screen:
		Enter date: 29-2-2001
		It is not a valid date.
