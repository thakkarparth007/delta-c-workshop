The math header file

	
This file contains a lot of functions for doing mathematical calculations. 
Here's a list of common functions that you might come across:
Function name Return type Arguments Example Explanation
abs int int abs(-8) returns absolute value (positive) of the argument
fabs double double fabs(-8.0) returns absolute value of the argument
sqrt double double sqrt(144.0) returns the square root of the given argument
exp double double exp(3) returns e3 - the exponentiation function
log double double log(10.3) returns loge10.3 - natural logarithm
sin double double sin(0.9) returns the sin of the given argument
cos double double cos(0.9) returns the cos of the given argument
tan double double tan(0.9) returns the tan of the given argument
asin double double asin(0.9) returns the sin-inverse (arcsin) of the given argument
cos double double cos(0.9) returns the cos-inverse (arccos) of the given argument
atan double double atan(0.9) returns the tan-inverse (arctan) of the given argument
