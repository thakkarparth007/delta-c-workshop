The float data type

This is one of the most common mistakes in C beginners.
They forget about integer approximation.

When you declare a variable as int we tell the program that it stores only integral values

Now, (25*35)/2 is not an integer. It has digits following the decimal point. 
If the variable is an int, those digits are ignored. 
 Data loss! 
Use:
1. Type casting
2. Use a higher data type than int -> A float [You just learnt a new data type]
