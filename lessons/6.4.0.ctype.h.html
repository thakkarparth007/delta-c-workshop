The ctype header file

	
This file contains a lot of functions for doing character manipulations. 
Here's a list of common functions that you might come across:

isupper, islower, toupper, tolower, isdigit, isalpha, isalnum
Function name Return type Arguments Example Explanation
isupper char int isupper('a') returns a non-zero number if the argument is an uppercase (capital) character. Otherwise 0.
islower char int islower('a') returns a non-zero number if the argument is a lowercase (capital) character. Otherwise 0.
toupper char int toupper('a') returns the corresponding uppercase character for a lowercase character. If the argument isn't a letter, then the same character is returned.
tolower char int tolower('A') returns the corresponding lowercase character for an uppercase character. If the argument isn't a letter, then the same character is returned.
isdigit char int isdigit('a') returns non-zero number if the argument is a digit. returns 0 otherwise.
isalpha char int isalpha('a') returns non-zero number if the argument is a letter. returns 0 otherwise.
isalnum char int isalnum('a') returns non-zero number if the argument is a digit or a letter. returns 0 otherwise.
ispunct char int ispunct('a') returns non-zero number if the argument is a punctutation. returns 0 otherwise.
isspace char int isspace('a') returns non-zero number if the argument is a whitespace (tab/space/new line/carriage-return). returns 0 otherwise.
